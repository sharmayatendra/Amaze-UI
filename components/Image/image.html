Images

Images are used to showcase the external form of anything be it person or anything else.

Amaze-UI has different kinds of images.

Responsive image

Use the "img-res" to the img tag image will grow to the parent container width.

Rounded image

Use "rounded-img" class in img tag the image will take circular shape with width of parent container.Look into code snippet for reference.
