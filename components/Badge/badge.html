Badge

A Badge is used to show the online status of the users by integrating it with avatars & can also be used for cart notification count.

Amaze-UI has two different kinds of badges.

Badge on Avatars

Badges can be used with or without count.Add corresponding class with "badge" class "large-size" , "online" etc as shown in the snippet

Badge on Icons

Badges on icons can be used simply by using "icon-badge" class with "badge" class.
