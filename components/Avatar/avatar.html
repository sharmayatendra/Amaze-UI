Avatar

An Avatar is used to show the display picture of the user.

Amaze-UI has different kinds of avatar.

Different Sized Rounded Avatars

Avatar component has 3 sizes large(6em),default(4em),small(2em).Just add corresponding class as shown in the example to img tag.

Rounded Avatars

Rounded avatars have size of default(4em).Just add class as "rounded-avatar" in img tag.
